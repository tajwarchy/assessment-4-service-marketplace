# Service Marketplace (Multi-Vendor Booking Platform)

A full-stack, multi-tenant service marketplace (Admin / Vendor / End-User roles),
built as part of Assessment 4 — Vibe Coding Challenge.

## Tech Stack
- Frontend: React + Vite + Tailwind CSS (Vercel)
- Backend: Node.js + Express + Prisma (Render)
- Database: PostgreSQL (Supabase)
- Auth: JWT + bcrypt, custom RBAC middleware
- Payments: Self-built sandbox payment simulator
